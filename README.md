# Система управления запасами для малого предприятия

## Описание
Прототип информационной системы для управления товарными запасами на малом предприятии. Система позволяет отслеживать остатки товаров, управлять поступлениями и списаниями, генерировать отчеты.

## Функциональные возможности
- Управление товарами (CRUD операции)
- Управление категориями товаров
- Операции поступления и списания товаров
- Отслеживание остатков на складе
- Оповещение о низких остатках
- Генерация отчетов

## Технологический стек
- Java 17
- SQLite (база данных)
- Maven (сборка)
- JUnit 5 (тестирование)
- Docker (контейнеризация)
- GitHub Actions (CI/CD)
